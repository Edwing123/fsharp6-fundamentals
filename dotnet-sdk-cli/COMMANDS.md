# Dotnet CLI commands

A list of some important Dotnet CLI commands.

```pwsh
# Get help information about the CLI.
dotnet --help

# Get the version of the installed Dotnet SDK.
dotnet --version

# List the installed SDKs.
dotnet --list-sdks

# List the installed runtimes(?).
dotnet --list-runtimes

# Create a new .NET project.
dotnet new

# Build a .NET project.
dotnet build

# Run a .NET project.
dotnet run

# Watch for changes on a .NET project and perform some actions upon that.
dotnet watch

# Invoke the F# Interactive REPL.
dotnet fsi
```
